Data build instructions
=======================

This application contains a large amount of embedded sound and image data.
These are contained in data.cpp and data.h, which were generated by a python script.
If changes need to be made to them, edit the data in data/ and run data.py to
regenerate these files.

Prerequisites

- Python 3
  Can be downloaded at http://www.python.org/

  
Win32 build instructions
========================

Prerequisites

- Visual Studio 2012 C++ Express
  Can be downloaded for free from Microsoft

- SDL-1.2.15 development libraries.
  Can be downloaded from http://www.libsdl.org/
  This was the latest version at time of writing, later versions will likely work.
  It was placed in a folder at the same level as the rest of the source,
  called SDL\ but this path can be changed in mariopants.vcxproj if needed.

- zlip-1.2.7
  Can be downloaded from http://www.zlib.net/
  This is placed in a folder at the same level as the rest of the source,
  called zlib\, and is built as part of the project.

After downloading and placing the prerequisites, you should
be able to open and build the mariopants.sln solution.


Mac OS X build instructions
===========================

Prerequisites

- XCode
  Can be downloaded for free from Apple

- SDL-1.2.15 development libraries.
  Can be downloaded from http://www.libsdl.org/
  This was the latest version at time of writing, later versions will likely work.
  This was placed in SDL/SDL-1.2.15 and the static libraries were built and installed to SDL/build.

- zlip-1.2.7
  Can be downloaded from http://www.zlib.net/
  This is placed in a folder at the same level as the rest of the source,
  called zlib\, and is built as part of the project.

After downloading and placing the prerequisites, build the XCode project in the xcode folder.
